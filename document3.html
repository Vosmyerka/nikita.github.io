Tailwind Elements

Tailwind CSS - это CSS-фреймворк с открытым исходным кодом. Главная особенность этой библиотеки заключается в том, что, в отличие от других CSS-фреймворков, таких как Bootstrap, она не предоставляет ряд предопределенных классов для таких элементов, как кнопки или таблицы. Вместо этого она создает список "полезных" CSS-классов, которые можно использовать для стилизации каждого элемента путем смешивания и подбора.

Namecheap
