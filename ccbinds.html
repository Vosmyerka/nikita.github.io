CCBinds

This program is designed to allow Convoy Control conveniently make and use multiple messages. The program is written entirely in the AutoHotKey language. Its principle of operation is that it simulates keystrokes to send messages to the game chat when user presses the key specified in the settings.

Instructions

  1. Go to settings and specify the keys you want to use
    1. On the left are keys that send only one predefined message when pressed
    2. On the right are keys that will send a message continuously until you press the key or key combination again
    3. Next to each key on the right, there is a button to press to save the key to a file
    4. All specified keys and key combinations are saved in a file called ccbinds.ini located in the Documents directory
  2. On the main page, enter the messages you want to remember in the input fields, then click the Start Script button, and the program will start working
  3. The program comes in two versions:
    1. Regular CCBinds.exe, which works ONLY in TruckersMP
    2. CCBindsNoWinActive.exe, which works everywhere
